Pi Visualizer

This pi estimation visualizer uses the Monte Carlo Method to estimate the value of pi. The program randomly generates points in a square with a circle inscribed into the square. Then pi is caluclated by multiplying 4 to the ratio between the number of points generated in the circle and the number of total points.
